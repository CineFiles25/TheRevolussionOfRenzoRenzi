Goal of the Project

Create structured, interoperable data starting from a cultural idea and a curated set of items related to it.

The final deliverables include:

  • CSV metadata for each item
  • TEI/XML encoding + HTML transformation
  • XML/TEI → RDF transformation
  • RDF dataset for the entire collection
  • A conceptual model based on existing ontologies
  • A website documenting the full process

Instructions (from course requirements)

1. Find the idea

We chose Renzo Renzi, critic, filmmaker, archivist and major promoter of film culture in Bologna. He was among the key figures who enabled the growth of the Cineteca after its founding in 1962.

2. Select 10+ items

  • different object types (books, photographs, drawings, audiovisuals, sound, documents)
  • at least one full-text item
  • possibly from different institutions

3. Metadata Analysis

Identify the metadata standards used by the institution holding each item.

4. Theoretical Model

Describe features of each item in natural language, based on institutional metadata + contextual knowledge.

5. Conceptual Model

Create a formal presentation of the domain using existing ontologies (RDF/RDFS, OWL, SKOS, DCTerms, Schema.org, FOAF, CIDOC-CRM, RiC-O, IFLA LRM). A graphical representation is required.

6. Create the deliverables

  • ✓ TEI/XML sample
  • ✓ XSLT transformation
  • ✓ XML → RDF transformation
  • ✓ RDF dataset
  • ✓ Website

Renzi and LOD: a decades-long history
Renzi assembled books, scripts, drawings, photographs, interviews, and film documents into an interconnected archive where each object illuminated the others — a network of film knowledge. The Renzi Collection can be read as an analogue prototype of Linked Open Data.
