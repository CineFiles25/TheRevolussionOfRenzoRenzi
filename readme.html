The “Revolussion” of Renzo Renzi

A Digital Humanities & Linked Open Data Project
Information Science and Cultural Heritage – University of Bologna (2024–2025)

Overview

The “Revolussion” of Renzo Renzi is a Digital Humanities project dedicated to the archival, bibliographic, and audiovisual materials related to Renzo Renzi (1919–2004), a key figure in the history of film culture in Bologna as critic, filmmaker, researcher, and curator.

The project investigates Renzi’s analogue “web of cinema knowledge” by modelling and interlinking a selection of materials from the Renzo Renzi Collection at the Cineteca di Bologna: books, photographs, drawings, sound recordings, films, and a video interview.

The project integrates:

  • TEI/XML scholarly text encoding
  • XSLT transformation for web publication
  • CSV-based metadata extraction and analysis
  • Linked Open Data modelling (RDF/Turtle)
  • Reuse of cultural heritage metadata standards
  • A conceptual model reusing existing ontologies
  • A web interface presenting all deliverables

The dataset includes 15 heterogeneous cultural heritage items (books, photographs, drawings, sound recordings, film materials, an audiovisual interview, institutional data), described, encoded, and semantically linked.

Objectives

Theoretical model

The theoretical model is documented as an interactive graph (Miro): open board.

  1. Define a structured dataset starting from a coherent cultural theme (Renzo Renzi and the network around La Strada and the Renzo Renzi Collection).
  2. Convert descriptive metadata into CSV using institutional standards (ISBD(G), ISBD(NBM), Scheda F, Scheda OA, FIAF rules, ISAD(G)).
  3. Encode one major item using TEI P5 and publish it through XSLT.
  4. Develop a conceptual model reusing: Dublin Core Terms, Schema.org, FOAF, SKOS (with RiC-O and CIDOC CRM used as conceptual references).
  5. Produce a complete RDF dataset where each selected object is exported as an individual Turtle file.
  6. Publish all deliverables in an accessible website built with GitHub Pages.

Website

The website presents:

  • TEI-encoded La Strada edition
  • Metadata tables and analysis
  • RDF dataset (modular Turtle files)
  • Image gallery
  • Team and acknowledgements
  • Documentation and downloads
